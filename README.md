## üß† Project Overview
The **Daily Calorie Tracker CLI** is a simple Python console-based tool that allows users to:
- Log their daily meals  
- Enter calorie values for each meal  
- Calculate **total** and **average** calorie intake  
- Compare against a personal daily calorie **limit**  
- Get an instant **warning** if they exceed their limit  
- (Bonus) Save the session details to a text file for record keeping  

This mini-project demonstrates **basic Python programming** using only **Unit 1 and Unit 2** concepts.
